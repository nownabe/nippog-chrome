nippog
======================
chrome拡張で動作する日報作成補助ツールです。
簡単なテキスト整形機能を持っています。

http://blog.nownabe.com/?p=162

インストール方法
------
1. [パッケージ](http://nownabe.com/~nownabe/nippog.crx)をダウンロードしてください
2. chromeのURL欄に`chrome://extensions`と入力し、拡張機能画面を開いてください
3. ダウンロードした`nippog.crx`を拡張機能画面にドラッグ&ドロップしてください

使い方
------
### 入力 ###
* 欄が足りなくなったら、`Add Section`ボタンとか`Add Chapter`ボタンとかをクリックしてください

### Preview ###
* 右上の`Preview`ボタンをクリックするとプレビューできます

### View Source ###
* 右上の`Source`ボタンをクリックするとHTMLのソースが見られます
* ソースウィンドウの`Hilight`ボタンをクリックするとソースが選択されるので、コピーとかできます

### Save ###
* 入力を一時保存できます
* ローカルストレージに保存できます

### Restore ###
* 一時保存したものを復元できます
* ChapterとかSectionとかを追加すると復元できません
* 復元すると、一時保存したものは消えます。また中断する場合は再度保存してください
